
# Simple Spring Boot Application With Sample Use ADRs and ArchUnit

## Pre-requisites

  - Maven
  - Java 11
  - Docker
  - Docker Compose

## How To Run

In _/contact_ runs:

```sh
$ docker-compose up
```
```sh
$ mvn clean install
```
```sh
$ mvn spring-boot:run
```
## Run Tests

```sh
$ mvn tests
```
